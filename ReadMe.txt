Запити до API:

Функція get_areas() отримує області.
Функція get_cities() отримує міста для кожної області.
Функція get_warehouses() отримує відділення для кожного міста.
Обробка даних:

Дані записуються в змінну all_data, яка організована за областями, містами та відділеннями.
Запис в Excel:

Використовується бібліотека openpyxl для створення Excel-файлу та запису даних по колонках (A: область, B: місто, C: відділення).
Перед використанням:

Замініть рядок API_KEY = 'your_api_key_here' на свій реальний API ключ.

Після виконання програми Excel-файл буде збережено в поточній директорії з назвою nova_poshta_data.xlsx.


Для встановлення 

Створення віртуального середовища virtualenv
python3 -m venv virtualenv

Активація віртуального середовища
source virtualenv/bin/activate

Встановлення бібліотек requests openpyxl
pip install requests openpyxl
